HTML 태그의 종류는 다음과 같습니다.

  • <h[1-6]> : 제목을 표시할 때 사용된다.
  • <p> : 새 문단을 연다.
  • <b> : 두껍게 효과를 준다.
  • <ins> : 밑줄을 표시한다.
  • <del> : 취소선을 표시한다.
  • <hr> : 가로줄을 넣는다.
  • <ul> : 순서 없는 목록을 표시한다.
  • <ol> : 순서 있는 목록을 표시한다.
  • <a> : 하이퍼링크를 생성하는 태그이다.
  • <img> : 페이지에 이미지를 추가하는 태그이다.
